* [Live Demo] (https://www.aotomot.com/aotomot-gallery-demo/)


== Installation ==


1. Upload the `aotomot-gallery` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
3. You can adjust the necessary settings using your WordPress admin panel in "settings" > "Aotomot Gallery".


== Frequently Asked Questions ==

= How to add gallery in WordPress post or page? =

You can add Aotomot gallery easily via shortcode `[aotomot-gallery]` or `[aotomot-gallery id="12"]`
